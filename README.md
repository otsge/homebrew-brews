# OTSGE Brews

## How do I install these formulae?

`brew install otsge/brews/<formula>`

Or `brew tap otsge/brews` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "otsge/brews"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
